# Seth
      Seth is the first Intelligent Proof-of-Work (iPoW) Layer-1 blockchain that integrates learned mining strategies with on-device AI inference.
      

# Quick Start
## Requirements
      gcc 15.2.0
      cmake3.25.1+

## Run local seth network
      git clone git@github.com:tenondvpn/SethChain.git seth
      bash build_third.sh
      bash simple_dep.sh $node_count  
      # node_count like 4, mean create 4 nodes seth network on local machine
	  
```
